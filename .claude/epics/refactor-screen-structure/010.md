# Task: Extract Chat Tab

## Context
Move all chat-related UI logic from the monolithic `chat_screen.dart` into a dedicated `ChatTab` widget.

## Description
Create `lib/screens/home/tabs/chat_tab.dart`. This file should contain a `StatefulWidget` (to manage `TextEditingController` and `ScrollController`) that implements the original `_ChatTabContent`, `ChatBubble`, `_StreamingIndicator`, and the input area helpers from `ChatScreen`.

## Implementation Steps
- Step 1: Create `lib/screens/home/tabs/` directory.
- Step 2: Define `ChatTab` as a `StatefulWidget`.
- Step 3: Copy `_controller`, `_scrollController`, and chat-related initialization/disposal logic from `_ChatScreenState`.
- Step 4: Implement `_handleSend`, `_scrollToBottom`, and other chat-specific helper methods within `_ChatTabState`.
- Step 5: Move `ChatBubble` and `_StreamingIndicator` classes into `chat_tab.dart`.
- Step 6: Update all imports and ensure `ChatProvider` is accessed correctly via `context.watch()` or `Consumer`.

## Acceptance Criteria
- [ ] **FR-2 / Happy path:** Chat messages can be sent and are displayed correctly in the new tab.
- [ ] **FR-2 / Streaming:** AI typing indicator and auto-scrolling work during message generation.
- [ ] **NFR-2 / Logic Parity:** All chat features (bubbles, record cards in chat) function exactly as before.

- Produces: `ChatTab` widget for `HomeScreen`.
