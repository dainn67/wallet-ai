# Task: Update ChatApiService for language awareness

## Context
The AI assistant needs to know the user's preferred language to provide accurate parsing and responses.

## Description
Modify `ChatApiService.streamChat` to dynamically include the active language in the request body `inputs`.

## Acceptance Criteria
- [ ] **FR-3 / Payload:** The `'language'` field in the API request body matches the user's selected language (e.g., "English" or "Vietnamese").

## Implementation Steps
1. Open `lib/services/chat_api_service.dart`.
2. Add a `String language` parameter to `streamChat` OR access it via `AppConfig`/`LocaleProvider`.
3. Update the `inputs` map:
   ```dart
   'inputs': {
     'language': currentLanguageName, // "English" or "Vietnamese"
     ...
   }
   ```
4. Update `ChatProvider` to pass the correct language from `LocaleProvider` when calling `streamChat`.

## Tests to Write
- Integration test: Verify the payload construction in `test/services/chat_api_service_test.dart`.
