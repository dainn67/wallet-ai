# Task: Define L10nConfig and initial translations

## Context
The app currently has hardcoded English strings. We need a central place to define supported languages, currencies, and their translations to support expansion.

## Description
Create `lib/configs/l10n_config.dart`. Define enums for `AppLanguage` (English, Vietnamese) and `AppCurrency` (USD, VND). Implement a `Map`-based translation system for core UI strings in both EN and VI.

## Acceptance Criteria
- [ ] **FR-1 / Configuration:** Enums for `AppLanguage` and `AppCurrency` are defined.
- [ ] **FR-1 / Translations:** A `Map<AppLanguage, Map<String, String>>` contains at least core strings for Drawer, Tabs, and Popups.
- [ ] **FR-1 / Currency Metadata:** Each currency has associated metadata (symbol, code).

## Implementation Steps
1. Create `lib/configs/l10n_config.dart`.
2. Define `enum AppLanguage { english, vietnamese }`.
3. Define `enum AppCurrency { usd, vnd }`.
4. Create `class L10nConfig` with:
   - `static const Map<AppLanguage, Map<String, String>> translations = { ... }`.
   - Add initial keys like `drawer_records`, `drawer_chat`, `reset_all_data`, `confirm_delete_source`.
   - `static const Map<AppCurrency, String> currencySymbols = { AppCurrency.usd: '$', AppCurrency.vnd: '₫' };`.
5. Export the new config in `lib/configs/configs.dart`.

## Tests to Write
- Unit test: `test/configs/l10n_config_test.dart` to verify all languages have the same set of keys.
