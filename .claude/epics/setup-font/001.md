# Task: Register Poppins Fonts & Verify Assets

## Context
The app needs to move from runtime fetched fonts to local assets for Poppins. This task handles the registration of those assets in the Flutter configuration.

## Description
Add the `fonts` section to `pubspec.yaml` and map all 18 Poppins TTF files from `assets/fonts/` to the `Poppins` family. Ensure each weight and style is correctly defined.

## Acceptance Criteria
- [ ] **FR-1 / Happy path:** `pubspec.yaml` contains a `fonts` entry for `Poppins` with all weights registered.
- [ ] **FR-1 / Verification:** `fvm flutter pub get` runs successfully without asset errors.

## Implementation Steps

### Step 1: Update pubspec.yaml
- Open `pubspec.yaml`.
- Find the `flutter:` section.
- Add/Update the `fonts:` sub-section:
  ```yaml
  fonts:
    - family: Poppins
      fonts:
        - asset: assets/fonts/Poppins-Thin.ttf
          weight: 100
        - asset: assets/fonts/Poppins-ThinItalic.ttf
          weight: 100
          style: italic
        - asset: assets/fonts/Poppins-ExtraLight.ttf
          weight: 200
        - asset: assets/fonts/Poppins-ExtraLightItalic.ttf
          weight: 200
          style: italic
        - asset: assets/fonts/Poppins-Light.ttf
          weight: 300
        - asset: assets/fonts/Poppins-LightItalic.ttf
          weight: 300
          style: italic
        - asset: assets/fonts/Poppins-Regular.ttf
          weight: 400
        - asset: assets/fonts/Poppins-Italic.ttf
          weight: 400
          style: italic
        - asset: assets/fonts/Poppins-Medium.ttf
          weight: 500
        - asset: assets/fonts/Poppins-MediumItalic.ttf
          weight: 500
          style: italic
        - asset: assets/fonts/Poppins-SemiBold.ttf
          weight: 600
        - asset: assets/fonts/Poppins-SemiBoldItalic.ttf
          weight: 600
          style: italic
        - asset: assets/fonts/Poppins-Bold.ttf
          weight: 700
        - asset: assets/fonts/Poppins-BoldItalic.ttf
          weight: 700
          style: italic
        - asset: assets/fonts/Poppins-ExtraBold.ttf
          weight: 800
        - asset: assets/fonts/Poppins-ExtraBoldItalic.ttf

          style: italic
        - asset: assets/fonts/Poppins-Black.ttf
  ```

### Step 2: Verify Assets
- Run `fvm flutter pub get` to ensure the paths are correct.
