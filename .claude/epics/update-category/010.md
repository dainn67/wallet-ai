# Task: CategoriesTab UI Scaffold & HomeScreen Integration

## Context
App cần thêm một không gian chuyên biệt để người dùng quản lý danh mục. Task này mở rộng thanh điều hướng TabBar và tạo khung màn hình CategoriesTab mới.

## Description
Cập nhật `HomeScreen` để hỗ trợ 3 tab chính: Chat, Records, và Categories. Khởi tạo file `CategoriesTab` với một `ListView` cơ bản hiển thị danh sách Categories từ `RecordProvider`.

## Acceptance Criteria
- [ ] **FR-2 / Tab Navigation:** `HomeScreen` hiển thị 3 TabBar icon: Chat, Records, Categories (thêm icon `Icons.category_outlined`).
- [ ] **FR-2 / Proper Indices:** Tab thứ 3 (index 2) mở màn hình `CategoriesTab`.
- [ ] **FR-2 / List Display:** `CategoriesTab` hiển thị danh sách các Category name hiện có.
- [ ] **FR-2 / Responsive Header:** Header tab Categories có nút "Thêm" (+) ở góc trên bên phải (ngang hàng với tiêu đề như ở Sources).

## Implementation Steps

### Step 1: Update HomeScreen
- Modify `lib/screens/home/home_screen.dart`
- Cập nhật `TabController` length từ 2 lên 3 (hoặc 3 lên 4 nếu ở dev mode).
- Thêm `Tab(icon: Icon(Icons.category_outlined), text: l10n.translate('drawer_categories'))` vào `TabBar`.
- Thêm `const CategoriesTab()` vào `TabBarView`.
- Cập nhật localization key `drawer_categories`.

### Step 2: Create CategoriesTab Scaffold
- Create `lib/screens/home/tabs/categories_tab.dart`
- Xây dựng một `StatelessWidget` (hoặc `StatefulWidget` nếu cần scroll controller).
- Sử dụng `Consumer<RecordProvider>` để lắng nghe danh sách categories.
- Sử dụng `Scaffold` với `AppBar` (nếu tab có appBar riêng) hoặc tích hợp header vào body.
- Thêm `ListView.separated` để hiển thị các `Category` item.
- Thêm nút `IconButton(icon: Icon(Icons.add), onPressed: ...)` ở đầu danh sách (trong header).

### Step 3: Add Localization Key
- Cập nhật các file l10n để thêm key `drawer_categories` (EN: 'Categories', VI: 'Danh mục').

## Technical Details
- **Architecture:** `CategoriesTab` là một tab con trong `HomeScreen`, thừa hưởng `Scaffold` chính.
- **Patterns to follow:** Xem cách `RecordsTab` và `ChatTab` được tổ chức trong `lib/screens/home/tabs/`.
- **Edge cases:** Xử lý trường hợp danh sách trống bằng một `EmptyState` widget thân thiện.
