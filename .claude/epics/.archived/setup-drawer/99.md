# Task: Drawer UI Cleanup and Version Layout

## Context
The drawer currently contains redundant navigation links that duplicate the `TabBarView` functionality.

## Description
Remove the navigation `ListTile` items and prepare the bottom of the drawer to display the app version.

## Implementation Steps
1. Open `lib/screens/home/home_screen.dart`.
2. Locate `_buildAppDrawer`.
3. Remove the `ListTile` widgets for 'Chat', 'Records', and 'Test'.
4. Wrap the `Column` in the `Drawer` with a `SafeArea` if necessary.
5. Add an `Expanded` spacer or use `MainAxisAlignment` to push a new `Padding` widget to the bottom.
6. Add a placeholder `Text` at the bottom (e.g., "v1.0.0(1)") with subtle styling (e.g., `Colors.grey`, fontSize 12).

## Acceptance Criteria
- [x] Navigation links (Chat, Records, Test) are gone from the drawer.
- [x] A version placeholder is visible at the very bottom of the drawer.
- [x] UI layout remains stable and looks clean.
