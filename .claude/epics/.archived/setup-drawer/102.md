# Task: Currency Toggle and Persistence

## Context
Users need to be able to switch between VND and USD, and this preference must be saved.

## Description
Add a currency toggle to the drawer and integrate it with `StorageService` and `CurrencyHelper`.

## Implementation Steps
1. Open `lib/services/storage_service.dart`.
2. Add constants for keys: `static const String keyCurrency = 'user_currency';`.
3. Open `lib/helpers/currency_helper.dart`.
4. Update `format` to accept an optional `currency` string or read it from `AppConfig`/`Storage`.
5. Update logic to handle USD (commas for thousands, dots for decimals) vs VND (dots for thousands, commas for decimals).
6. Open `lib/screens/home/home_screen.dart`.
7. Add a `ListTile` or `SwitchListTile` to the drawer for Currency.
8. On change, save the value to `StorageService` and trigger a UI refresh (e.g., via `setState` or a provider if available).

## Acceptance Criteria
- [ ] Drawer has a functional toggle for VND and USD.
- [ ] Changing the currency updates the formatted amounts in the UI (if wired).
- [ ] The choice is persisted after app restart.
