# Task: AppConfig Version Support

## Context
The app version needs to be hardcoded in `AppConfig` so it can be accessed by the UI without adding heavy dependencies.

## Description
Add `version` and `buildNumber` fields to the `AppConfig` singleton.

## Implementation Steps
1. Open `lib/configs/app_config.dart`.
2. Add `final String version = '1.0.0';` (use current pubspec value as default).
3. Add `final String buildNumber = '2';` (use current pubspec value as default).
4. (Optional) Add a getter `String get fullVersion => 'v$version($buildNumber)';`.

## Acceptance Criteria
- [x] `AppConfig` has `version` and `buildNumber` fields.
- [x] Fields are accessible via the singleton instance.
- [x] App compiles correctly.
