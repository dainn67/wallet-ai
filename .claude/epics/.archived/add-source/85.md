# Task: Add Source Button to RecordsOverview

## Context
The `RecordsOverview` component displays a list of sources but lacks a way to add new ones. This task adds a "+" button next to the "Sources" title to trigger the creation flow.

## Description
Modify `lib/components/records_overview.dart` to add an `IconButton` with `Icons.add_rounded` next to the "Sources" title text. The button should be aligned to the trailing end of the title row.

## Implementation Steps
1. Open `lib/components/records_overview.dart`.
2. Locate the "Sources" title text widget (around line 75).
3. Wrap the title `Text` and the new `IconButton` in a `Row` with `mainAxisAlignment: MainAxisAlignment.spaceBetween`.
4. The `IconButton` should:
   - Use `Icons.add_rounded`.
   - Have a small size (e.g., 18-20).
   - Use `Colors.white.withValues(alpha: 0.6)` to match the title style.
   - On press, it should call a placeholder function for now (to be updated in T011/T012).
