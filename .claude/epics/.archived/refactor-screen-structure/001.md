# Task: Scaffold Home Screen

## Context
Create the new `HomeScreen` which will serve as the primary navigation container for the app, replacing the current `ChatScreen`'s layout role.

## Description
Create `lib/screens/home/home_screen.dart`. It should implement a `StatefulWidget` that uses a `DefaultTabController` with 3 tabs (Chat, Records, Test). The UI should include the `AppBar`, `Drawer`, and `TabBarView` currently found in `chat_screen.dart`. For now, use `Placeholder()` widgets for the `TabBarView` children.

## Implementation Steps
- Step 1: Create the directory `lib/screens/home/` if it doesn't exist.
- Step 2: Create `home_screen.dart` and copy the `Scaffold`, `AppBar`, and `Drawer` logic from `ChatScreen`.
- Step 3: Implement `DefaultTabController` with `length: 3`.
- Step 4: Add `TabBar` to `AppBar.bottom` with icons: `chat_bubble_outline`, `receipt_long`, `science_outlined`.
- Step 5: Implement `TabBarView` in the `body` with three `Placeholder()` widgets.

## Acceptance Criteria
- [ ] **FR-1 / Happy path:** `HomeScreen` renders with a working `TabBar`.
- [ ] **FR-1 / Navigation:** Swiping or tapping tabs changes the active (placeholder) content.

## Verification
- Run `fvm flutter run` and verify the new `HomeScreen` renders correctly (once integrated).
