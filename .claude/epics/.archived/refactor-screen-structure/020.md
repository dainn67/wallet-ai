# Task: Integration and Cleanup

## Context
Integrate the extracted tabs into `HomeScreen`, update the app's entry point, and remove the deprecated monolithic `chat_screen.dart`.

## Description
This task involves replacing `Placeholder()` widgets in `HomeScreen` with `ChatTab`, `RecordsTab`, and `TestTab`. It also includes updating `lib/screens/screens.dart` to export `HomeScreen`, updating `lib/main.dart` to use `HomeScreen` as the default home, and finally deleting the original `chat_screen.dart`.

## Implementation Steps
- Step 1: In `lib/screens/home/home_screen.dart`, import and replace the `Placeholder()` widgets with `ChatTab()`, `RecordsTab()`, and `TestTab()`.
- Step 2: Update `lib/screens/screens.dart` to export `'home/home_screen.dart'` and remove the `chat_screen.dart` export.
- Step 3: In `lib/main.dart`, change the `MaterialApp`'s `home` property from `ChatScreen()` to `HomeScreen()`.
- Step 4: Delete `lib/screens/chat_screen.dart`.
- Step 5: Fix any broken imports throughout the project (especially in tests).

## Acceptance Criteria
- [ ] **FR-3 / Happy path:** The app launches directly into the new `HomeScreen`.
