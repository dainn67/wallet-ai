# Task: Update Record Card UI

## Context
Adding a timestamp to each record card allows users to see the exact date of a transaction at a glance within the list.

## Description
Modify the `RecordWidget` to include a small, subtle timestamp in `dd/mm/yyyy` format. This involves using the `intl` package (added in T001) to format the `record.createdAt` field and placing the result cleanly in the UI, likely in the bottom-right corner or alongside the subtitle.

## Acceptance Criteria
- [x] **FR-1 / Happy path:** Each record card displays a formatted date (e.g., "21/03/2026").
- [x] **FR-1 / Styling:** Timestamp is legible, uses Poppins font, and is subtly colored (e.g., `Colors.grey`).
- [x] **FR-1 / Layout:** Timestamp does not overflow or interfere with existing description and amount text.

## Implementation Steps

### Step 1: Add Date Formatting logic
- [x] Open `lib/components/record_widget.dart`.
- [x] Import `package:intl/intl.dart`.
- [x] Add a helper method or format directly in the `build` method:
  `final formattedDate = DateFormat('dd/MM/yyyy').format(DateTime.fromMillisecondsSinceEpoch(record.createdAt));`

### Step 2: Update UI Layout
- [x] Add a `Text` widget displaying `formattedDate` within the record card.
