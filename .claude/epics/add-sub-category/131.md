# Task: AI Prompt Hierarchical Formatting

## Context
Update the AI service to include the parent-sub structure in its system prompt to ensure accurate classification.

## Interface Contract
- **Receives from T130**: Correctly seeded database with hierarchy.

## Acceptance Criteria
- [ ] `ChatApiService.formatCategories` builds hierarchical strings: `ID: [SubName] (Category: [ParentName])`.
- [ ] AI correctly receives this list in the prompt.
- [ ] AI correctly maps transaction descriptions to sub-category IDs.

## Implementation Steps
1.  **Modify `lib/services/chat_api_service.dart`**:
    *   Find the `formatCategories` method.
    *   Update logic to iterate over the categories.
    *   For each sub-category (`parentId != -1`), find its parent's name.
    *   Format the string as: `${c.categoryId}-${c.name} (Parent: ${parentName})`.

## Verification
- Test `ChatApiService.formatCategories` with a mock list of categories.
- Perform an integration test (manual chat) and verify the AI classifies to a sub-category (e.g., "burger" → "Restaurant").
