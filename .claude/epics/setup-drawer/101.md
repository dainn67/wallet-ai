# Task: Python Version Sync Script

## Context
We need a way to keep `pubspec.yaml` and `AppConfig` versions in sync automatically during the build process.

## Description
Create a Python script that parses a version string and updates both the YAML file and the Dart config file.

## Implementation Steps
1. Create directory `scripts/` if it doesn't exist.
2. Create `scripts/update_version.py`.
3. The script should:
    - Accept one argument: `x.y.z+b`.
    - Use regex to find and replace `version: ...` in `pubspec.yaml`.
    - Use regex to find and replace `version = '...'` and `buildNumber = '...'` in `lib/configs/app_config.dart`.
    - Provide clear output on what was updated.
4. Add basic error handling for missing files or invalid format.

## Acceptance Criteria
- [x] Running `python3 scripts/update_version.py 1.1.0+5` updates `pubspec.yaml` to `version: 1.1.0+5`.
- [x] The same command updates `AppConfig` to `version = '1.1.0'` and `buildNumber = '5'`.
- [x] Script handles the `+` separator correctly.
