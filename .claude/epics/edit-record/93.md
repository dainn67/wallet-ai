# Task: Create EditRecordPopup UI

## Context
A UI is needed to allow users to modify existing records.

## Description
Create `lib/components/popups/edit_record_popup.dart` with a dark theme consistent with the app's style.

## Implementation Steps
1. Create `lib/components/popups/edit_record_popup.dart`.
2. Implement `EditRecordPopup` as a `StatefulWidget` accepting an existing `Record`.
3. Fields to include:
    - Amount (TextField with numeric keyboard)
    - Type (ToggleButtons or Switch for Income/Expense)
    - Money Source (Dropdown/Selector)
    - Category (Dropdown/Selector)
    - Description (TextField)
4. Use `BorderRadius.circular(28)` for the dialog and `12` for inputs.
5. Initialize all fields with the values from the passed `Record`.

## Acceptance Criteria
- [ ] Popup displays correctly with all required fields.
- [ ] Initial values match the record being edited.
- [ ] UI matches the dark theme (0xFF1E293B background).
- [ ] "Cancel" and "Save" buttons are present.
