# Task: Update Record Model & Repository Queries

## Context
Link records to categories in the code and database queries.

## Description
1. Update the `Record` model to include `int categoryId` and an optional `String? categoryName`.
2. Update `RecordRepository.getAllRecords()` to perform a SQL `LEFT JOIN` with the `Category` table to fetch category names.
3. Update `Record.fromMap` to handle the `category_name` field from the joined query.

## Implementation Steps
- Step 1: Add `categoryId` and `categoryName` to `Record` class and constructor.
- Step 2: Update `Record.fromMap` and `Record.toMap`.
- Step 3: Rewrite `getAllRecords` query in `RecordRepository` to use `JOIN`.
