# Task: Create AddSourcePopup UI

## Context
The app requires a popup to capture new source details. This task implements the UI component for that popup, following the app's established dark theme.

## Description
Create a new file `lib/components/popups/add_source_popup.dart` and implement a `StatefulWidget` named `AddSourcePopup`. It should present a dialog with two input fields: Source Name and Initial Amount.

## Implementation Steps
1. Create the directory `lib/components/popups/` if it doesn't exist.
2. Create `add_source_popup.dart`.
3. Use a `Dialog` or `Container` with `BorderRadius.circular(28)` for the main container.
4. Background should use `Color(0xFF1E293B)` or `Color(0xFF0F172A)`.
5. Add two `TextFields`:
   - "Source Name": Standard text input.
   - "Initial Amount": Use `keyboardType: TextInputType.numberWithOptions(decimal: true)`.
6. Use `InputDecoration` with `BorderRadius.circular(12)`.
7. Add "Cancel" and "Save" buttons.
8. Use `Navigator.of(context).pop()` for closing and returning data.

## Acceptance Criteria
- [ ] **FR-2 / scenario:** A popup dialog appears with input fields for Name and Initial Amount.
- [ ] **NFR-1 / UI Consistency:** The popup matches the app's dark theme and uses specified border radiuses (28px container, 12px inputs).

## Tests to Write
- Widget test to verify the presence of "Source Name" and "Initial Amount" fields in the dialog.
