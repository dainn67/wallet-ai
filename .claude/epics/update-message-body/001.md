# Task: Service Formatting Helpers

## Context
Provide helper methods to format app data into a simple string format for AI prompts.

## Description
Add static methods to `ChatApiService` to format `List<MoneySource>` and `List<Category>` into `id-Name` strings (e.g., "1-Bank, 2-Wallet").

## Implementation Steps
- Step 1: Add `static String formatMoneySources(List<MoneySource> sources)` to `ChatApiService`.
- Step 2: Add `static String formatCategories(List<Category> categories)` to `ChatApiService`.
- Step 3: Ensure the output format is clean and compact (e.g., comma-separated).

## Acceptance Criteria
- [x] **FR-1 / Formatting:** Helper methods return strings in the `id-Name` format.
- [x] **FR-1 / robustness:** Handle empty lists gracefully.

## Verification
- Unit test or print log the output of these helpers with sample data.
